Paitulin OGC-rajapintapalvelut

Monet Paitulin aineistot ovat saatavilla OGC:n mukaisista rajapintapalveluista. OGC rajapintapalvelut mahdollistavat datan latauksen suoraan netti-, työpöytäsovellukseen tai aineiston analyysiskriptiin, näin että dataa ei ladata ensin paikalliselle koneelle tiedostona. Tämä mahdollistaa aina ajan tasalla olevan datan käytön. Rajapintapalvelun käyttö on työpöytäsovelluksissa samanlainen kuin paikallisella levyllä olevan datan käyttö, rajapintapalveluun otetaan yhteyttä vastaavasta valikosta. Yhteyden muodostamiseksi tarvitaan palvelun URL osoite. Yhteyden muodostamisen jälkeen käyttäjälle listataan kaikki palvelusta saatavilla olevat karttatasot.

Saatavilla olevat aineistot

Paituli tarjoaa yli 600 aineistoa OGC rajapintojen kautta. Aineistojen tarkat nimet löytyvät Latauspalvelusta: valitse aineisto ja sitten klikkaa Rajapinnat.

Uudet OGC API rajapinnat

OGC API rajapinta-standardit ovat vielä kehittymässä. Paitulissa OGC API rajapintojen saatavuus riippuu GeoServerin tarjottavista mahdollisuuksista.

Käyttövinkkejä

WMTS-palvelussa karttatiilet ovat saatavilla vain etukäteen määritetyissä mittakaavoissa ja laajuuksissa. WMS-palvelussa kartan koko ja mittakaava voidaan valita vapaasti. WMTS-palvelu on yleensä nopeampi, sillä karttatiilet ovat usein jo palvelun välimuistissa valmiina. Karttatasosta voidaan pyytä myös vain osa käyttämällä suodattimia, esim. BBOX määrittele pyydettävän maantietteellisen alueen.

WMTS-palveluiden ruudukot: ERTS-TM35-FIN (EPSG:3067, JHS180) ja Spherical Mercator (EPSG:3857).

Rajapinta-aineistojen lisenssiehdot ovat vastaavat kuin latauspalvelun kautta ladatuissa aineistoissa. Lisenssiehdot löytyvät Latauspalvelu sivulta.

Jos Paitulin rajapintoja käytetään nettisovelluksessa, aineiston tuottaja, nimi (mm vuosi) ja Paituli on mainittava.
